# Book My Show

## Requirement

### APIs

1. list all movie in a city
2. list all theatre in a city
3. given a show display all seats
4. book tickets (show detail, selected seats)
5. cancel booking 
6. Add movie, theatre
7. Add/Delete shows

### Additional information

1. Seats can be of different types
    - silver, gold and platinium
2. Show can be of different types
3. Different mode of payment
4. One seat cannot be booked by two customers for same show
5. Every show-seat has a base fair admin should be able to increase the fair of shows based on some predefined rules
